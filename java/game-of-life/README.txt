installing java
 - go to https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html
 - click Accept License Agreement
 - download link that corresponds to the .exe for your version of Windows
 - install jdk
 - add the full path of the jdk-11\bin directory to the PATH variable

installing maven
- download binary zip from http://maven.apache.org/download.cgi
- unzip it to some folder
- add a MAVEN_HOME system variables, and point it to the Maven folder
- add %MAVEN_HOME%\bin to the PATH variable
